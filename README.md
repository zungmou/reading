# reading

阅读是一种享受，是一种生活方式，是一种态度。

> 仅以此项目献给我的家人

# Features

- [√] 语音朗读
- [√] 拼音显示
- [×] 评论互动
  - [disqus](https://disqus.com/)
- [×] 文章审核
- [×] 全文搜索
  - [Google Custom Search](https://programmablesearchengine.google.com/) *（国内无法使用）*
  - [lunr.js](https://lunrjs.com/) *（需要自行构建索引）*

- [edge-tts](https://github.com/rany2/edge-tts)
